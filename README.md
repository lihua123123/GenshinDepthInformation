# 原神产球及附着计算器

一个用于查看原神角色技能元素附着规则和产球机制的在线工具。

## 📋 项目简介

本项目提供了原神游戏中所有角色的技能元素附着量、附着规则、产球数量等详细数据，帮助玩家更好地理解和优化队伍配置。

## 📁 文件结构

```
原神计算器/
├── README.md                        # 项目说明文档
├── 元素附着及产球.md                 # 数据源文件（Markdown格式）
├── 原神产球及附着.html               # 完整版HTML文件（单文件，可直接打开）
└── 原神产球及附着拆分/                # 模块化版本（HTML+CSS+JS分离）
    ├── index.html                   # 主页面结构
    ├── style.css                    # 样式文件
    └── script.js                    # 脚本文件
```

## ✨ 功能特点

### 1. 元素筛选

- 支持按元素类型筛选：火、水、雷、冰、风、岩、草
- 实时切换显示不同元素的角色数据

### 2. 武器类型筛选

- 支持按武器类型筛选：单手剑、双手剑、长柄武器、弓、法器
- 可与元素筛选组合使用

### 3. 详细数据展示

每个角色显示以下信息：

- **技能名称**：普攻、重击、E技能、Q技能、命座等
- **元素量**：弱/强/超强元素附着
- **附着规则**：附着冷却时间、独立判定等
- **产球数量**：元素微粒/元素晶球产生数量
- **备注**：特殊机制说明
- **抗打断**：技能的抗打断等级

### 4. 界面特性

- 🌓 **深色/浅色模式切换**：支持夜间模式，保护眼睛
- 📱 **响应式设计**：完美适配桌面、平板、手机等设备
- 🎨 **元素量颜色标识**：不同强度的元素附着使用不同颜色显示
- 🔍 **粘性导航栏**：滚动页面时筛选栏始终可见
- ⚡ **性能优化**：流畅的切换动画和交互体验

## 🚀 使用方法

### 方式一：使用完整版（推荐快速使用）

直接用浏览器打开 `原神产球及附着.html` 即可使用，无需额外配置。

### 方式二：使用模块化版本（推荐开发/修改）

1. 打开 `原神产球及附着拆分/index.html`
2. 如需修改样式，编辑 `style.css`
3. 如需修改功能，编辑 `script.js`

### 操作指南

1. **选择元素**：点击顶部的元素按钮（火、水、雷等）切换查看不同元素角色
2. **筛选武器**：点击武器类型按钮进行二次筛选
3. **切换主题**：点击右上角的月亮/太阳图标切换深色/浅色模式
4. **查看数据**：浏览角色表格，查看各项技能数据

## 📊 数据说明

### 元素量等级

- **弱元素**：基础元素附着（绿色）
- **中元素**：中等元素附着（黄绿色，仅草元素）
- **强元素**：增强元素附着（蓝色/红色等）
- **超强元素**：最强元素附着（深色）

### 附着规则

- **独立**：拥有独立的附着判定
- **X s/Y hit**：每X秒或Y次攻击触发一次附着
- **共用CD**：与其他技能共享附着冷却时间

### 产球机制

- 显示每个技能产生的元素微粒/晶球数量
- 影响队伍充能效率的重要参数

## 🔄 数据维护

### 更新流程

1. 在 `元素附着及产球.md` 中更新角色数据
2. 运行同步脚本更新HTML文件（或手动更新）
3. 数据格式要求：

   ```markdown
   ## 角色名
   | 技能 | 元素量 | 附着规则 | 产球 | 备注 | 抗打断 |
   |------|--------|----------|------|------|--------|
   | 内容 | 内容   | 内容     | 内容 | 内容 | 内容   |
   ```

### 版本同步

- 完整版和拆分版的数据保持同步
- 更新时需同时维护两个版本
- 建议使用脚本自动同步
