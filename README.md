# 原神产球及附着计算器

原神角色技能元素附着规则和产球机制查询工具，包含116+角色完整数据。

## ✨ 主要功能

- **元素筛选**：按火、水、雷、冰、风、岩、草七大元素分类查看
- **武器筛选**：按单手剑、双手剑、长柄武器、弓、法器筛选角色
- **角色充能计算器**：输入角色名称自动填充能量，计算循环充能要求
- **属性强化表**：圣遗物词条强化区间和平均值查询
- **普攻产球表**：各武器类型的普攻产球概率和平均攻击次数
- **深浅色模式**：支持主题切换，保护视力

## 📋 角色数据内容

每个角色包含以下信息：
- 技能名称（普攻、重击、E技能、Q技能、命座等）
- 元素附着量（弱/中/强/超强元素）
- 附着规则（冷却时间、独立判定等）
- 产球数量（元素微粒/晶球）
- 抗打断等级
- 特殊机制说明

## 🚀 使用方法

### 完整版
直接打开 `原神产球及附着.html` 即可使用。

### 模块化版本
打开 `原神产球及附着拆分/index.html`，方便自定义修改样式和功能。
